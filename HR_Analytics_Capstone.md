# Data Analysis Capstone Project: HR Analytics

## Project Overview

Welcome to your final capstone project! You have been hired as a **Junior Data Analyst** for a growing corporate enterprise. The HR Director has handed you an export from their legacy HR system. Unfortunately, the data is incredibly messy, inconsistent, and currently unusable for decision-making.

Your objective is to apply everything you've learned — from basic data cleaning and advanced formulas to Pivot Tables and interactive Dashboard creation — to transform this raw data into a compelling, automated HR Dashboard.

---

## Phase 1: Data Cleaning & Standardization

Go to the **`Raw_Employee_Data`** sheet and perform the following cleaning tasks:

- Remove all duplicate records from the dataset.
- Standardize the columns and format each column with its correct data type.
- Handle missing values the best way you can.
- Clean and match values using the lookup table.

---

## Phase 2: Data Transformation & Advanced Formulas

Create the following new columns to enrich your dataset:

| Column | Description |
|---|---|
| **Full Name** | Combine first and last name fields into one column |
| **Department Name** | Resolve department codes to full department names |
| **Years of Service** | Calculate duration from hire date to today |
| **Performance Band** | Categorize employees by performance score range |
| **Eligible Bonus Amount** | Calculate projected bonus based on salary and performance band |

---

## Phase 3: Exploratory Data Analysis (Pivot Tables)

Create a new worksheet named **`Analysis`** and insert Pivot Tables to answer the following business questions:

1. **Headcount & Salary** — What is the total headcount and average salary per Department?

2. **Attrition Rate** — What is the Attrition Rate by Department? *(Calculate the percentage of 'Left' employees vs. total employees per department.)*

3. **Performance by Status** — How does the average Performance Score differ between Active and Left employees?

4. **Bonus Payout** — What is the total Bonus Payout projected for this year, broken down by Performance Band?

---

## Phase 4: Dashboard Creation

Create a final worksheet named **`HR Dashboard`**. Your dashboard must include:

- A clean, professional layout *(remove gridlines and use a consistent color palette)*.
- At least **3 KPI Cards** at the top — e.g., Total Employees, Average Salary, Overall Attrition Rate.
- Relevant charts for your visuals.
- **Interactive Elements** — Add Slicers and connect them to all charts so the dashboard is fully dynamic.
